Welcome, Eileen

This short Microsoft Office prep sequence is designed to help you feel more comfortable using Excel and PowerPoint for college assignments, projects, and business-school coursework.

You do not need to learn every feature in Excel. Most college tasks rely on a practical set of skills used again and again: entering information, organizing it clearly, formatting it professionally, using basic formulas, creating simple charts, and saving or submitting files correctly.

Formula reminders: Start with =, use cell references, use : for a range, and always check whether the answer makes sense.

Common Excel Mistakes to Watch For

  • Forgetting the = sign at the start of a formula.
  • Selecting the wrong cells in a formula.
  • Typing numbers as text.
  • Forgetting to format percentages.
  • Copying a formula but not checking the changed references.
  • Saving the file with an unclear name or submitting the wrong version.

Session 1: Excel Foundations

Session product: College Budget Tracker

Main goal: Create and save a clean first spreadsheet while practicing basic Excel navigation, formatting, and formulas.

Today’s big idea: Excel becomes much less intimidating when you focus on a few core moves: enter information, make it readable, use formulas to calculate, and save your work clearly.

What You Will Be Able to Do

  • Open a blank Excel workbook.
  • Identify rows, columns, cells, worksheets, and the formula bar.
  • Enter and edit data.
  • Adjust column widths and format headers.
  • Format numbers as currency.
  • Use a subtraction formula.
  • Use =SUM().
  • Save a file with a clear name.
  • Explain what the spreadsheet does.
